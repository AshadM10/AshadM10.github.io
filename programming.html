Variables

In order to remember something and come back to it later, store it in a variable. You do this by typing var name = "Ashad". You can refer to it afterwards by just typing name.

Types of variable

You can have lost of different types of variable, the most common ones are:

  • String : (word, sentence or URL) several letter "strung" together. You can tell if something is a string because it has "inverted commas" around it.
  • Integer : Whole number
  • Long : Decimal number
  • Boolean : True or false.
